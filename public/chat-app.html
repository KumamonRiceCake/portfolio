Chat app

Github Link: https://github.com/KumamonRiceCake/node-chat-app/

App on Heroku: https://kim-chat-application.herokuapp.com/

Features

This node.js application provides a chat service to users.

Users can create open chat rooms for anyone or secret rooms secured with a password.

The message information is delivered using socket.io.

Users can send their location information using Geolocation API and Google Map.

Quality assured with integration test using Jest.


How to use

On the main page, users can see active room names and the number of participants in each room.

If a room is locked with a room password, it includes ðŸ”’ icon next to the room name.

To create or join a room, enter a user name, room name, and room password, and then press enter or click the join button.

If a room password is not given, the room is not locked.

Once users enter a chat room, they can see the names of participants.

Users can send text messages and their location information.

Self-messages appear on the right side with yellow text boxes, while received messages are in white boxes on the left side.
