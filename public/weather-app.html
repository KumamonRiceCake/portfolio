Weather app

Github Link: https://github.com/KumamonRiceCake/weather-forecast-nodejs

App on Heroku: https://kim-weather-forecast-app.herokuapp.com/

Features

This simple node.js application lets users search weather information of their current location or desired location.

It requests data from location API provided by https://www.mapbox.com/ and weather API provided by https://weatherstack.com/.


How to use

To search weather information of the desired location, type search term in the text box and click the 'Search' button or press enter key.

To see weather information of current location, click the 'My location' button.
